This tool was originally created for Hristo's Loyal Bison Slugs - my trivia team in Halifax established in 2022. Files that you load into this game rendered completely by your browser; nothing is transferred from your computer.

Instructions

Use this template to populate trivia questions.

Download Template

Each tile requires a unique question and answer and must have a unique point value within its category. You can add more questions within the category, but only four categories - any more and it will not load properly.

This is a work in progress. I hope to make it so that you can include media such as URLs for images or embed codes for youtube videos. If you are not using media, keep those cells blank.
