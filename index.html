This tool was originally created for Hristo's Loyal Bison Slugs - my trivia team in Halifax established in 2022. Files that you load into this game rendered completely by your browser; nothing is transferred from your computer.

Instructions

Use this template to populate trivia questions.

Download Template

Setup:

  • Category headings must be unique. You can have up to four categories.
  • Point values must be numbers and must be unique for each question/answer/media combination.
  • Media is optional. To include it, provide the complete url to an image file. If you don't want media, leave those cells blank.
